# Map Algebra Model Lab

It's the best.

### Dependencies

- Scala 2.11
- Apache Spark 2.1.0+

### Getting Started

To use MAML, add the following to your `build.sbt`:

```
libraryDependencies += "com.azavea" %% "maml" % "1.0.0"
```

Now import the following, and you're good to go:

```tut:silent
// import maml.ast.jvm._
```
